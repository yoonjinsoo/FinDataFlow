Objective: Create a professional, single-page landing page using TypeScript and Tailwind CSS. The page is for a B2B purchase financing service targeting established online sellers. The primary goal is to qualify leads through a self-checklist and then allow them to submit an inquiry via a mailto: action.

Core Concept: The service's identity is a "trusted growth financing partner," not a simple loan company. The design should feel like a modern FinTech platform: professional, trustworthy, clean, and data-driven.

3. Design System & Styling (with Tailwind CSS)

Font: Use a modern, clean sans-serif font like Pretendard.

Color Palette:

Primary: A deep, trustworthy Navy Blue (#1E3A8A) or Charcoal Gray (#374151).

Accent: A vibrant, intelligent Tech Blue (#3B82F6) for all CTA buttons, links, and active states.

Neutral: White (#FFFFFF) and a very light gray (#F9FAFB) for backgrounds. Text should be a dark gray (#111827).

Layout: Use ample whitespace for a premium, uncluttered feel. Maintain a consistent grid and spacing system.

Components:

Buttons: rounded-md, clear hover and focus states. Disabled buttons should have a lower opacity (opacity-50) and cursor-not-allowed.

Forms: Inputs should have clean borders that change color on focus (to the Accent color).

Cards: Use subtle shadows (shadow-md) and rounded corners on card-like elements to create depth.
